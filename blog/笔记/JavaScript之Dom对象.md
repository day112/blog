## 文档树（DOM树）

`DOM`即文档对象模型（Document Object Model）。它使用对象的表示方式来表示对应的文档结构及其中的内容。

### HTML转换DOM树

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My title</title>
  </head>
  <body>
    <a href="http://www.baidu.com">My Link</a>
    <h1>My header</h1>
  </body>
</html>
```
