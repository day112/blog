## Momentjs常用API



### 解析日期

#### 1. 获取当前时间

```js
moment();
moment(undefined);
// 从v2.14开始也支持以下格式
moment([]);
moment({});		
```

#### 2. 字符串日期

> moment(String);



| Input      | Example          | Description                                       |
| :--------- | :--------------- | :------------------------------------------------ |
| `YYYY`     | `2014`           | 4或2位数的年份                                    |
| `YY`       | `14`             | 2位数的年份                                       |
| `Y`        | `-25`            | 有任意数字和符号的年份                            |
| `M MM`     | `1..12`          | 月份                                              |
| `MMM MMMM` | `Jan..December`  | 根据月份名称设置月份，例如May。 `moment.locale()` |
| `D DD`     | `1..31`          | 一个月中的某天                                    |
| `X`        | `1410715640.579` | Unix 秒级时间戳                                   |
| `x`        | `1410715640579`  | Unix 毫秒级时间戳                                 |
