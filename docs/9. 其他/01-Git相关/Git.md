#### Git配置123123

```shell
$ git config --global  # 对当前用户的所有仓库有效
$ git config --system  # 对所有用户的所有仓库有效
$ git config --local   # 仅对当前仓库有效

$ git config --global user.name "hua03"
$ git config --global user.email "huahao093@gmail.com"
```
